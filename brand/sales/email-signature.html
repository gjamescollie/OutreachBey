How to use this signature

  1. Open this file in your browser.
  2. Select everything inside the signature box below (the table area).
  3. Copy it (Cmd+C / Ctrl+C).
  4. In Gmail: Settings → General → Signature → paste into the signature editor.
  5. In Apple Mail: Mail → Settings → Signatures → paste.

The signature uses inline styles and a table layout for maximum email-client compatibility.
